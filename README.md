# 🚀 Remindly

> Never lose an opportunity again.

Remindly is a full-stack application built to help students and developers save, organize, and keep track of everything that matters—internships, hackathons, project ideas, articles, useful links, deadlines, and personal notes.

The idea came from a simple problem: important opportunities are scattered across WhatsApp, LinkedIn, X (Twitter), YouTube, Discord, and browser bookmarks. We save them with the intention of coming back later, but most of the time they're forgotten.

Remindly aims to solve that by bringing everything together in one place.

---

## ✨ Features

- 🔐 Secure user authentication
- 📌 Save opportunities, links, and notes
- 🗂️ Organize items with categories and tags
- ⭐ Mark important items as favorites
- 🔍 Powerful search and filtering
- ⏰ Deadline reminders and notifications
- 📊 Dashboard with insights and statistics
- 🌙 Light & Dark mode
- 📱 Available on both Web and Mobile

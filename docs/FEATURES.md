# Features

## Vision

Remindly is a personal opportunity management platform designed to help users save, organize, search, and revisit important opportunities before they are forgotten.

Instead of relying on browser bookmarks, notes, chats, or screenshots, users can manage everything in one centralized application accessible from both web and mobile.
---
# User Authentication
Users can:
- Register a new account
- Log in securely
- Log out
- Stay authenticated using JWT
- Update their profile
- Change their password
